cppreference: "cpp/io/c/puts"
type: "C-style I/O Library"
token: "<cstdio>"
---

provides C-style I/O operations.
It is also part of the C standard library as `<stdio.h>`.

Example usage:

```cpp notip=true
#include <cstdio>

auto main() -> int {
  std::puts("Hello world!");
  std::printf("Formated print %d!\n", 42);
}
```
